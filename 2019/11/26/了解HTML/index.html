了解HTML

在一个以.html或.htm后缀的文件中写入上面的代码都是可行的，没有区别。

HTML不是编程语言，是标记语言，是一套标签(markup tag)，用来描述网页。

HTML文档包含HTML标签及文本内容，又称WEB页面。

1、文档类型声明

HTML5

HTML 4.01

XHTML 1.0

查看完整网页声明类型 DOCTYPE 参考手册。

2、标签

HTML标签大小写不敏感，但推荐全部小写，因未来的XHTML强制全部小写。

文档标题：<title></title>

页面标题：<h1h6></h1h6>

段落：<p></p>

链接：<a href=””></a>

图像：<img src=”” width=”” height=””></img>

空标签：没有内容的标签，如<br>换行，<hr>水平线标签

注释：<!–注释内容–>

粗体“bold”：<b></b>

斜体“italic”：<i></i>

<strong> 替换<b>或者 <em>替换<i>意味着你要呈现的文本是重要的，所以要突出显示。

小号字：<small></small>

上下标字：<sup><\sup> <sub></sub>

插入字：<ins></ins>

删除字：<del></del>

预定义格式文本：<pre></pre>

3、属性

给HTML标签提供属性，为标签添加附加信息，在开始标签中描述这些属性。

属性值始终用引号引起，双引号是最常用的，特殊情况下需要单双引号一起使用。

推荐属性和属性值小写，XHTML强制要求小写。

下面列出了适用于大多数 HTML 元素的属性：

属性 描述
class 为html元素定义一个或多个类名（classname）(类名从样式文件引入)
id 定义元素的唯一id
style 规定元素的行内样式（inline style）
title 描述了元素的额外信息 (作为工具条使用)

4、<head>标签，定义文档信息

可以添加在头部区域的元素标签为: <title>, <style>, <meta>, <link>, <script>, <noscript>, and <base>.

<base> 标签描述了基本的链接地址/链接目标，该标签作为HTML文档中所有的链接标签的默认链接

<link> 标签定义了文档与外部资源之间的关系。

<style> 标签定义了HTML文档的样式文件引用地址.

<mate> META 元素通常用于指定网页的描述，关键词，文件的最后修改时间，作者，和其他元数据。

<script>标签用于加载脚本文件，如： JavaScript。

5、CSS样式

内联样式-HTML标签中

内部样式-<style>标签中

外部样式-使用外部CSS文件

标题很重要

请确保将 HTML 标题 标签只用于标题。不要仅仅是为了生成粗体或大号的文本而使用标题。

搜索引擎使用标题为您的网页的结构和内容编制索引。

因为用户可以通过标题来快速浏览您的网页，所以用标题来呈现文档结构是很重要的。

应该将 h1 用作主标题（最重要的），其后是 h2（次重要的），再其次是 h3，以此类推。

请始终将正斜杠添加到子文件夹。假如这样书写链接：href=”https://www.runoob.com/html"，就会向服务器产生两次 HTTP 请求。这是因为服务器会添加正斜杠到这个地址，然后创建一个新的请求，就像这样：href=”https://www.runoob.com/html/"。
