Java多态

什么是多态?

简单理解 : 看成一种事物多种形态

Animal a1 = new Human();

a1是人？是动物？还是Bee？它的类型是什么？

  • 运行时类型：对象的真正类型，从上面代码来看就是等号‘=’右边的类型；
  • 编译时类型：在编译的阶段，编译器在意的类型是等号 ‘=’ 左边的类型；
  • 当编译时类型和运行时类型不一致时—>产生多态。

Animal a = new Human();
Human p = new Human();
多态的写法的好处 ： 可以屏蔽不同的子类之间的实现差异

字段没有覆写一说：实际开发中一般不会再子类中定义一个和父类同名的字段
